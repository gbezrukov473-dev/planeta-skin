# Использование шаблонов Header/Footer

## Как это работает

Плагин `vite-plugin-html-template.js` автоматически вставляет содержимое `templates/header.html` и `templates/footer.html` в HTML файлы во время сборки.

## Использование в HTML файлах

### Вариант 1: Использование плейсхолдеров (рекомендуется)

В вашем HTML файле замените существующие блоки header и footer на плейсхолдеры:

```html
<body>
  <!-- HEADER_PLACEHOLDER -->
  
  <!-- Ваш контент страницы -->
  <main>
    ...
  </main>
  
  <!-- FOOTER_PLACEHOLDER -->
</body>
```

### Вариант 2: Ручная замена (для существующих файлов)

Если у вас уже есть header/footer в HTML файлах, вы можете:

1. Удалить существующие блоки
2. Добавить плейсхолдеры `<!-- HEADER_PLACEHOLDER -->` и `<!-- FOOTER_PLACEHOLDER -->`

## Обновление шаблонов

Чтобы изменить header или footer на всех страницах:

1. Отредактируйте `templates/header.html` или `templates/footer.html`
2. Пересоберите проект: `npm run build`

## Преимущества

- ✅ Один источник правды для header/footer
- ✅ Легко обновлять на всех страницах одновременно
- ✅ Меньше дублирования кода
- ✅ Меньше вероятность ошибок при копировании

## Миграция существующих файлов

Для миграции существующих HTML файлов:

1. Найдите блок `<header>` и замените на `<!-- HEADER_PLACEHOLDER -->`
2. Найдите блок `<footer>` и замените на `<!-- FOOTER_PLACEHOLDER -->`
3. Убедитесь, что скрипт подключен в footer (он уже есть в шаблоне)

## Примечания

- Плагин работает только во время сборки (`npm run build`)
- В режиме разработки (`npm run dev`) изменения в шаблонах требуют перезапуска сервера
- Для более сложных шаблонов можно использовать переменные (требует доработки плагина)
